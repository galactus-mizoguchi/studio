# GALACTUS STUDIO デザイン変更案

## 基本方針

### 1. グラデーションの削除
- **現状**: ロゴ、タイトル、カード背景、アイコンなどでグラデーションを使用
- **変更後**: 単色ベースのデザインに統一
  - ロゴ: 単色の背景色（プライマリカラー）
  - タイトル: グラデーションテキスト → 単色テキスト
  - カード背景: グラデーション → 単色または微細なパターン
  - アイコン背景: グラデーション → 単色

### 2. 絵文字の削除
- **現状**: プロジェクトカードとアイデアリストで絵文字を使用
- **変更後**: 
  - プロジェクトカード: アイコンをSVGアイコンまたはテキストベースの記号に変更
  - アイデアリスト: 絵文字を削除し、テキストのみまたはシンプルなアイコンに変更

## デザインガイドラインに基づく改善

### タイポグラフィ強化
- **巨大サイズのタイトル**: サイトタイトルをより大きく、大胆に
- **Kinetic Typography**: スクロール連動アニメーションの追加
- **文字を主役に**: 視覚的な装飾を減らし、タイポグラフィで表現

### レイアウト改善
- **Bento Gridスタイル**: アイデアセクションをBento Grid風のレイアウトに
- **非対称配置**: 一部の要素を非対称に配置して動きを出す
- **重ね合わせ**: カード同士を軽く重ねて奥行きを表現

### マイクロインタラクション
- ホバー時の微細なアニメーション
- カードのリフト効果をより洗練
- トランジションのタイミングを最適化

## 具体的な変更内容

### HTML構造の変更
1. **プロジェクトカードのアイコン**
   - `<div class="project-icon">📱</div>` → `<div class="project-icon">R</div>` (ResponsiveのRなど)
   - またはSVGアイコンを使用

2. **アイデアリスト**
   - `<span class="idea-icon">🌐</span>` → 削除
   - テキストのみで表現、またはシンプルな記号（•、→など）

### CSS変更
1. **グラデーション変数の削除・置換**
   ```css
   /* 削除 */
   --gradient-primary: linear-gradient(...);
   --gradient-card: linear-gradient(...);
   
   /* 追加 */
   --color-primary-solid: hsl(260, 85%, 60%);
   --color-card-bg: #ffffff;
   ```

2. **ロゴスタイル**
   - グラデーション背景 → 単色背景
   - シンプルなシャドウで立体感を表現

3. **タイトルスタイル**
   - グラデーションテキスト → 単色テキスト
   - より大きなフォントサイズ
   - 文字間隔の調整

4. **カードスタイル**
   - グラデーション背景 → 単色背景
   - ホバー時のグラデーションオーバーレイ → 単色のオーバーレイまたはシャドウ

5. **アイコンスタイル**
   - グラデーション背景 → 単色背景
   - よりシンプルなデザイン

6. **背景エフェクト**
   - グラデーションオーブ → 単色のオーブまたは削除
   - より控えめな表現

## レイアウト提案

### プロジェクトグリッド
- 現状: 均等なグリッド
- 提案: Bento Gridスタイル
  - 一部のカードを大きく（2列分）
  - 高さも可変にしてリズムを作る

### アイデアセクション
- 現状: 均等なグリッド
- 提案: Bento Gridスタイル
  - カテゴリごとにグループ化
  - サイズを変えて視覚的な階層を作る
