# ResponsiveChecker - レスポンシブデザインチェッカー

URLを入力するだけで、複数のデバイスサイズでWebサイトの表示を確認できるツールです。

## 機能

- **複数デバイス対応**: iPhone SE、iPhone 12/13、iPhone 14 Pro、iPad、iPad Pro、デスクトップなど、主要なデバイスサイズでプレビュー可能
- **カスタムサイズ**: 任意の幅・高さを指定してプレビュー
- **スクリーンショット**: プレビュー画面を画像としてダウンロード
- **複数デバイス比較**: 複数のデバイスサイズを同時に比較表示
- **リアルタイムプレビュー**: iframeを使用した実際のページ表示

## 使い方

1. **URLを入力**: 確認したいWebページのURLを入力フィールドに入力
2. **プレビュー**: 「プレビュー」ボタンをクリックしてページを読み込み
3. **デバイス選択**: デバイスタブをクリックして表示サイズを切り替え
4. **比較に追加**: デバイスタブをCtrl/Cmd+クリックまたは右クリックで比較に追加
5. **スクリーンショット**: 「スクリーンショット」ボタンで画像をダウンロード

## デバイスサイズ

- **iPhone SE**: 375 × 667px
- **iPhone 12/13**: 390 × 844px
- **iPhone 14 Pro**: 393 × 852px
- **iPad**: 768 × 1024px
- **iPad Pro**: 1024 × 1366px
- **デスクトップ**: 1920 × 1080px
- **カスタム**: 任意のサイズを指定可能

## 注意事項

- 一部のサイトはセキュリティ設定により、iframeでの表示が制限されている場合があります
- CORS（Cross-Origin Resource Sharing）の制限により、一部のサイトが表示できない場合があります
- スクリーンショット機能は、html2canvasライブラリを使用しています
