# Web制作ヒアリングAI

クライアントの回答から、Webサイトの要件定義書とサイト構成・原稿のたたき台をAIで自動生成するミニアプリです。

## 機能

- **ヒアリングフォーム**: 7つの質問に回答するだけで要件定義書を生成
- **AI生成**: OpenAI APIを使用して要件定義書を自動生成
- **結果表示**: 生成された要件定義書を分かりやすく表示
- **コピー機能**: 生成結果をテキストとしてコピー可能

## 使い方

### 1. OpenAI APIキーの設定

1. [OpenAI](https://platform.openai.com/)でAPIキーを取得
2. ヒアリングフォームを開くと、APIキーの入力が求められます
3. 入力したAPIキーはlocalStorageに保存されます（開発用）

**注意**: 本番環境では、APIキーをクライアント側に保存しないでください。バックエンドサーバー経由でAPIを呼び出すことを推奨します。

### 2. ヒアリングの実施

1. `index.html`を開く
2. 「ヒアリングを開始」ボタンをクリック
3. 7つの質問に順番に回答
4. 最後に「生成開始」ボタンをクリック

### 3. 結果の確認

1. 生成が完了すると、自動的に結果ページに遷移
2. 生成された要件定義書を確認
3. 「コピー」ボタンでテキストをコピー可能

## ヒアリング質問

1. **業種・サービス内容**: どのような事業を行っているか
2. **主なターゲット**: サイトの主な訪問者（年齢層、法人・個人など）
3. **サイトの目的**: 問い合わせ、集客、採用など
4. **競合サイトURL**: 参考にしたいサイト（任意）
5. **伝えたい強み**: 3つまで選択可能
6. **希望ページ数**: おおよそのページ数
7. **トーン・雰囲気**: 信頼感、親しみ、高級感など

## 技術スタック

- **HTML5**: セマンティックなマークアップ
