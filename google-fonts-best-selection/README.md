# Google Fonts ベストセレクション

Webデザインでよく使われる定番・人気のGoogle Fontsを厳選して紹介するWebサイトです。

## 概要

このプロジェクトは、Google Fontsの中でも特に人気のあるフォントを日本語、サンセリフ、セリフ、筆記体の4つのカテゴリーに分類して紹介しています。各フォントには以下の情報が含まれています：

- フォント名とカテゴリー
- フォントの説明
- サンプルテキスト
- Google Fonts の読み込みコード
- Google Fonts へのリンク

## ファイル構成
